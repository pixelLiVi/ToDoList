# ToDoList

Eine moderne Task-Management-Anwendung built mit **Blazor** (.NET 10).

## Features

- ✅ Aufgaben erstellen
- ✅ Interaktive Benutzeroberfläche mit Bootstrap

## Anforderungen

- .NET 10 SDK oder höher
- Visual Studio 2022 oder VS Code

## Installation & Start

```bash
git clone https://github.com/pixelLiVi/ToDoList.git
cd ToDoList
dotnet restore
dotnet run
```

Öffne dann `https://localhost:7150` in deinem Browser.

## Struktur

```
ToDoList/
├── Components/
│   ├── Pages/          # Seiten-Komponenten
│   ├── Layout/         # Layout & Navigation
│   └── App.razor       # Root-Komponente
├── Program.cs          # Anwendungs-Setup
└── wwwroot/            # Statische Assets
```
